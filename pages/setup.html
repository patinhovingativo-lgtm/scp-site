HOW TO PUT THE SITE ONLINE

Follow these steps in order. The site will be at https://username.github.io/scp-site — accessible by anyone.

WHY file:// DOESN'T WORK FOR OTHERS: When you open the HTML file directly from your computer, it uses file:///C:/... — that path only exists on YOUR computer. Nobody else can access it. You need to host it online. GitHub Pages is free and takes 5 minutes.
STEP 1 — CREATE A GITHUB ACCOUNT

If you don't have a GitHub account:

  1. Go to github.com
  2. Click Sign up
  3. Create account with your email
  4. Verify email
Already have GitHub? Skip to Step 2.
STEP 2 — CREATE A REPOSITORY
  1. On GitHub, click the + button (top right) → New repository
  2. Repository name: scp-site
  3. Set to Public ✅ (required for free GitHub Pages)
  4. Do NOT check "Add README" — leave everything empty
  5. Click Create repository
STEP 3 — UPLOAD THE SITE FILES

Inside the ZIP you downloaded, there is a site/ folder. You need to upload the contents of that folder (not the folder itself).

  1. Open the repository you just created
  2. Click "uploading an existing file" (small link in the middle of the page)
  3. Extract the ZIP on your computer first
  4. Open the scp-v5/site/ folder
  5. Select ALL files and folders inside site/ and drag them to the GitHub upload area
  6. Wait for upload to finish
  7. Scroll down, click Commit changes
IMPORTANT: Upload what's INSIDE site/ — not the site/ folder itself. The index.html must be at the ROOT of the repository.
STEP 4 — ENABLE GITHUB PAGES (MAKES IT HTTPS)
  1. In your repository, click Settings (top tab)
  2. In the left sidebar, click Pages
  3. Under "Source": select Deploy from a branch
  4. Branch: main | Folder: / (root)
  5. Click Save
  6. Check "Enforce HTTPS" ✅
  7. Wait 2-5 minutes
  8. Your site is now at: https://YOUR-USERNAME.github.io/scp-site
This is the link you share with people! It works on any device, any browser, anywhere in the world.
STEP 5 — DEPLOY THE BOT ON RAILWAY

The bot handles applications, Discord login, and Roblox promotions. It needs to run 24/7 on a server.

  1. Create a second GitHub repository called scp-bot
  2. Upload the contents of the bot/ folder to it
  3. Go to railway.app → sign up free
  4. New Project → Deploy from GitHub repo → select scp-bot
  5. Railway detects Node.js automatically
  6. Go to Variables tab → add all these:

  2. Add a Volume mounted at /data (for database persistence)
STEP 6 — CONFIGURE THE ADMIN PANEL
  1. Open your site: https://your-username.github.io/scp-site/admin/index.html
  2. Enter your Railway bot URL (e.g. https://scp-bot.railway.app)
  3. Enter your ADMIN_API_KEY password
  4. Enter your Discord and Roblox usernames (optional extra security)
  5. Click ACCESS ADMIN PANEL
  6. Go to Roblox Ranks tab → click "Fetch from Roblox" → map your rank IDs
  7. Go to Config tab → set your site URL → click Save
STEP 7 — CONFIGURE DISCORD LOGIN (OAuth2)

So members can login with Discord on your site:

  1. Go to discord.com/developers/applications
  2. Click your bot application → OAuth2
  3. Under Redirects → click Add Redirect
  4. Enter: https://your-username.github.io/scp-site/verify.html
  5. Save
  6. Your members can now login at /verify.html
Members open /verify.html, enter the bot URL once, then click "Login with Discord". After that it's automatic.
