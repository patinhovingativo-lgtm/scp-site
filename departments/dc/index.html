READ BEFORE PROCEEDING: All experiments require proper authorization. Shortcuts will result in disciplinary action under COE Art. 3.4.
PHASE 1 — WRITE YOUR RESEARCH PROPOSAL

Before anything else, write a Research Proposal containing:

FIELDDESCRIPTIONEXAMPLE
SCP SubjectWhich SCP you will testSCP-173
ObjectiveWhat you want to discoverReaction to auditory stimuli
D-Class RequiredHow many and what condition2 D-Class, no injuries
MaterialsEquipment neededAudio equipment, blindfold
ProcedureStep-by-step planSee steps below
Risk LevelSafe / Euclid / KeterEuclid
Security NeededNumber of escorts2 Security Officers
PHASE 2 — GET APPROVAL

Submit your proposal to a Chief Researcher or above for sign-off. They will approve or request modifications.

After approval is granted, post in the department channel: "Research Proposal [ID] approved by [Researcher name]. Requesting resources."
PHASE 3 — REQUEST SECURITY ESCORT

Contact the Security Department and request an escort. Specify:

  • Number of D-Class needed
  • SCP object class (Safe/Euclid/Keter)
  • Estimated test duration
  • Your clearance level
RADIO FORMAT: "Security, this is Researcher [name], SC-[level]. Requesting [N] security escorts for a [class] experiment with SCP-[number]. D-Class count: [N]. ETA: [time]."

If no Security available: You may request MTF or DLOT assistance. They must still receive authorization from their commanding officer before escorting.

PHASE 4 — D-CLASS BRIEFING ROOM

Before proceeding to the SCP containment zone, Security must take D-Class to the Briefing Room and read the following rules:

MANDATORY D-CLASS RULES (Security reads aloud):
1. You will follow all instructions without question.
2. Attempting to escape will result in immediate termination.
3. Any hostile action will result in immediate termination.
4. You are participating in a Foundation research operation.
PHASE 5 — WEAPONS SCANNER

After the briefing room, D-Class must pass through the weapons scanner.

SCANNER RESULTACTION
CLEARD-Class may proceed to the testing area.
ARMED DETECTEDImmediate termination. An armed D-Class attempting to enter a test is considered hostile. No warnings. Terminate and document.
REASONING: An armed D-Class in a test environment is an unacceptable security risk. A prisoner who refuses to surrender their only means of escape in a controlled environment has demonstrated clear hostile intent. Termination is mandatory.
PHASE 6 — SCP RETRIEVAL / SETUP

Procedure depends on SCP type:

SCP TYPECLASSPROCEDUREWHO MOVES IT
OBJECTSafeAlready contained — bring D-Class to itResearcher + Security
OBJECTEuclid/KeterContainment Unit moves to testing roomDS Containment Unit (or Security if unavailable)
HUMANOIDSafeAlready contained — test in containment cellResearcher + Security
HUMANOIDEuclidMTF escorts SCP to designated testing roomMTF required
HUMANOIDKeterTest only in containment cell — SCP does NOT moveNo movement — ever
If no Containment Unit member available: Request any Security Officer. The Security Officer must request authorization from their commanding officer via radio or chat before proceeding. Reason: some SCPs require specialists.
PHASE 7 — CONDUCTING THE TEST

You are in command of the experiment. Security follows your instructions within their ROE. Document everything in real-time:

  • Note all D-Class reactions
  • Note all SCP behaviors and anomalous effects
  • Take screenshots/recordings of key discoveries
  • If unexpected breach occurs — abort immediately, evacuate D-Class, call Security
ABSOLUTELY PROHIBITED: Terminating D-Class without cause · Deliberately breaching other SCPs · Leaving D-Class alone with Euclid+ entities · Exceeding approved test parameters without re-authorization.
PHASE 8 — SUBMIT TEST LOG & DOCUMENT

After the test, submit a Test Log to the department channel within 24 hours. Include:

SECTIONCONTENT
HeaderSCP number, date, researcher name, clearance level
ObjectiveWhat you were testing and why
ProcedureStep-by-step of what happened
ResultsWhat was discovered — include screenshots
ConclusionWhat this means for containment/understanding
CasualtiesD-Class terminated, Security injured, etc.
Anomalous EffectsAny unexpected behavior observed
APPROVED LOGS COUNT TOWARD PROMOTIONS. Quality documentation is how researchers advance in rank.
DEPARTMENT RULES
#RULE
01 All tests require a written proposal approved by Chief Researcher or above.
02 D-Class may not be terminated during a test without documented cause.
03 Euclid-class testing requires a Security escort at all times.
04 Keter-class testing requires SC-3 clearance and two Security escorts minimum.
05 Test logs must be submitted within 24 hours of test completion.
06 Researchers may not perform experiments on themselves.
07 Any anomalous effect on a researcher must be reported immediately to Medical.
08 Inactive researchers (14+ days without LOA) will be demoted one rank.
09 Fabricating test results is a Foundation-wide violation (COE Art. 4.7). Instant dismissal.
