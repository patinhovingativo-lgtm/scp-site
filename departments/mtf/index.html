UNIT DESIGNATIONS

Each MTF squad operates under a designated unit code assigned by Operations Chief. Unit codes are used in all radio communication. Never use real names on active operations.

DEPLOYMENT CONDITIONS

MTF deploys under the following conditions: (1) Keter-class containment breach, (2) Hostile external faction incursion, (3) Mass D-Class uprising, (4) Direct O5 or Site Director order. MTF does NOT deploy for standard Security calls.

RULES OF ENGAGEMENT

✦ Level 1 — Non-lethal preferred: Standard trespass, low-risk D-Class.
✦ Level 2 — Warning before lethal: Armed hostile, escaping D-Class near civilians.
✦ Level 3 — Shoot on sight: Keter-class breach, external armed faction, O5 authorization.

BREACH RESPONSE PRIORITY ORDER

1. Protect O5 and Site Command · 2. Evacuate civilian departments · 3. Contain the anomaly · 4. Neutralize hostile personnel · 5. Secure and document the zone.
